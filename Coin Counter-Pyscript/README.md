# README

### A program to count the minimum number of coins required to provide a given amount of change.
The programme then lists the coins required to provide this minimum make-up. 
The programme assumes coins are provided in the following denominations: £2, £1, 50p, 20p, 10p, 5p, 2p, 1p.
