# Readme
## Card Picking Game

### Overview
This is a simplified CLI game written as part of a learning task. It replicates a simple version of a card-picking game (like simplified blackjack), but with a basic 20-card deck (2 sets of cards labelled 1 to 10), as opposed to the standard 52-card deck.

### Instructions
You play two rounds in which you pick two random cards from the deck (1 to 10). Each card can be selected only once and there are no duplicates.
Your opponent, "Computer" does the same. The aim is to pick cards totalling more than Computer's cards, with the sum of your cards still less that 21. 

By the second round, you will be able to see both of your cards and Computer's first card. You can then decide whether to play a third round, in which both you and Computer will pick an additional card. 
Otherwise, you can end the game, at which point the played hands are compared and a winner is declared.
