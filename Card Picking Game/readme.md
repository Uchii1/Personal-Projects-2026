# Readme

### Overview
This is a simplified CLI game written as a learning task. It replicates a more simple version of a card-picking game, but with a basic 20-card deck (2 sets labelled 1 to 10), as opposed to the standard 


### Instructions
You play two rounds in which you pick two random cards from the deck (1 to 10).
Your opponent, "Computer" does the same. The aim is to have cards picked totalling more than Computer's cards, with the sum of your cards still less that 20. 
By the second round, you will be able to see both your	 cards and Computer's first card. 
You can then decide whether to play a third round, in which both you can the computer will pick an additional card. 
Otherwise, you can end the game, at which point the played hands are compared and a winner is declared.
